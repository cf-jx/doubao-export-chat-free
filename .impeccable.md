## Design Context

### Users
Doubao web users who want to export a current conversation or a batch of captured conversations quickly.
They use the panel as a lightweight utility, not as a dashboard, and expect to finish the task with minimal decisions.

### Brand Personality
Clean, mature, direct.
The interface should feel stable, quiet, and productized instead of experimental or overly technical.

### Aesthetic Direction
Light desktop utility panel with restrained hierarchy, soft mint accents, and minimal motion.
Primary focus stays on choosing an export format and completing the export with minimal friction.

### Design Principles
1. Keep the default view focused on title, format, and export.
2. Show optional capability as inline controls, not as competing cards.
3. Hide implementation detail unless it is required for the user to act.
4. Preserve clear visual hierarchy: one primary choice, one primary action.
