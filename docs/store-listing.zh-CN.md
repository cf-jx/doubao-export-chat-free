# Chrome Web Store 中文资料草稿

## 基础信息

- 名称：豆包导出助手
- 简短说明：在浏览器本地将豆包网页对话导出为 Markdown、HTML、JSON 或 TXT。
- 语言：中文
- 价格：免费
- 分类建议：Workflow & Planning

## 详细说明

豆包导出助手用于备份本人豆包网页对话。打开豆包网页后，点击页面右侧的导出按钮，即可将当前对话导出为 Markdown、HTML、JSON 或 TXT。

适合这些场景：

- 将重要对话保存为本地文档。
- 用 Markdown 整理笔记。
- 用 HTML 保存可阅读网页副本。
- 用 JSON 保留结构化数据，便于后续处理。
- 用 TXT 保存纯文本内容。

主要功能：

- 当前对话导出。
- 导出消息时间。
- 按日期范围导出。
- 大对话自动分片，降低浏览器下载失败概率。
- 本地处理，不上传聊天记录到服务器。

隐私说明：

本扩展仅供用户本地备份本人账号聊天记录。导出处理在当前浏览器本地完成，扩展不会把聊天记录上传到服务器。

## 权限说明

- `storage`：保存导出设置、缓存和运行状态。
- `downloads`：保存用户主动导出的文件。
- `unlimitedStorage`：支持大对话和图片导出的本地缓存。
- 豆包域名访问权限：只在豆包网页中注入导出面板并读取当前网页对话数据。

## 测试说明

1. 安装扩展。
2. 打开 `https://www.doubao.com/` 并登录。
3. 打开任意对话。
4. 点击页面右侧的豆包导出按钮。
5. 选择导出格式。
6. 点击“导出当前对话”，确认浏览器下载文件。
