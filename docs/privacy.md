# 豆包导出助手隐私政策

生效日期：2026-07-09

豆包导出助手用于在浏览器本地备份本人豆包网页对话。

## 数据处理

- 扩展会在豆包网页中读取当前对话内容，用于生成用户主动导出的文件。
- 扩展会在浏览器本地保存导出设置、运行状态和必要缓存。
- 扩展不会把聊天记录、导出内容或个人身份信息上传到服务器。
- 扩展不提供账号系统，也不做跨设备同步。

## 权限用途

- `storage`：保存导出设置、缓存和运行状态。
- `downloads`：保存用户主动导出的文件。
- `unlimitedStorage`：支持大对话和图片导出的本地缓存。
- 豆包域名访问权限：只在豆包网页中运行导出面板并读取当前网页对话数据。

## 用户控制

用户可以在 Chrome 扩展管理页卸载扩展。卸载扩展会删除扩展本地存储的数据。用户也可以在浏览器下载目录中自行管理导出的文件。

## 联系与更新

隐私政策如有更新，会随仓库和发布页面同步更新。
